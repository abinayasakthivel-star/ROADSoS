ROADSoS 🚨

ROADSoS is a smart road safety and emergency assistance system developed for the IIT Madras Road Safety Hackathon. 
The project helps users access emergency support, nearby hospitals, and roadside assistance during accidents and emergency situations.

Features:
Emergency assistance support
Nearby hospital access
Roadside assistance
User-friendly interface
Accident response support

Technologies Used:
HTML
CSS
Bootstrap
JavaScript
Python
Flask
SQL
